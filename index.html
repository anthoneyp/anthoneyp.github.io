Posted on October 1, 2020

我对于未发生的事情都具有极强的控制欲，哪怕是无力改变的事情也想要尽可能多的思考到每一种可能会发生的情况，有些事即使我知道自己在做无用功。很累，但却依旧执着去做，会一直坚持到事情盖棺定论前一瞬间为止。我自己太缺乏自信，觉得自己处理不好即将到来的每一件事，害怕犯错，害怕担责，才寄希望于做好应对每一种情况的准备用于掩盖自己的无能。这本身就是一种无能。虽然在开始及过程我对此感到满足，觉得自己掌握着一切，有能力应对每一种情况。但最后临近结果时所有一切都会转为内心的无力及无助感，因为知道自己又做了无用功。结果真的重要吗？预测这么多又有什么用呢？能改变吗？我对得到结果感到平淡，发现生活的一切都没有乐趣。

大一生活零零散散，学业无成。

即将进入的大二

汽车+火车总共坐了大概14h，拖着重重的行李与背包在昨天早上八点才到学校，身体已经是疲惫不堪 但是又还得换宿舍(从原来宿舍搬到另一幢宿舍，而且搬到六楼，无电梯)，XX后勤处。截止到刚才我才安置好自己的所有东西，只是想吐槽下这个学校实在是太大了。

将旧宿舍的东西搬到新宿舍楼下后就没什么力气了，加上十几个小时没吃东西，都感觉要上天了，很怕自己突然死掉所以赶紧把手机密码改成爸妈都知道的数字（生日）。趴在桌子上休息了好一会。然后差不多快到中午的时候精神清醒了一点后给外公外婆打电话，报平安，刚听到声音后眼泪就控制不住流下来了。他们照顾了我十多年，对我来说是最善良最重感情的人。特别是听到外婆的声音后，有那么一段时间竟忍着不掉眼泪也说不出一句话。我第一次觉得自己是那么脆弱，而自己平时又装着很乐观向上的样子生活，想想大人们也都是这样的吧，都是因为这样大人们才会变老的吧。所以啊，小时候什么也不懂的时候，想着自己要去哪个哪个大城市去生活，想着离开现在的家庭，讲着自己长大后要做什么成为什么，那些东西现在看来都是多么幼稚，多么无理的梦啊。大人们那时候也都知道的吧，知道我们大多数人的未来，知道将要面对的现实与生活。但他们还是愿意相信且接受这一切，因为这些都是我们埋藏心底最朦朦胧胧的梦。

希望大二能成为我生命中的一个分水岭，希望家人平平安安，希望自己不忘初心。

大三

这一年我终于搬出校外住了，算起来我这三年换了三个居住的环境。从开始的桃源宿舍到桂苑，再到骆家塘。与某些室友的不合导致了我这几年都是郁郁寡欢，心理状态已经非常差了，不愿再去接触更多人，不再想交真心的朋友。对于这样的结果我也不想解释太多，如果非要对这些事情做个总结的话，就都算我的错吧。那天到达金华已经是晚上的十点半了，在假期时候我在网上已经找好了房子，旧的宿舍我不想再多待一刻。（在写这些文字我会开始不由自主的烦躁，对于不想要想起的事情我总是很排斥，我尽量克制）。那天我来到公寓楼下后联系房东开门，我还是第一次见可以手机远程开门的门禁，我好土。房东也很惊讶为何我执着于在快晚上十一点的时候马上入住，但她并没有问太多，还是很友善地给我开了门，交代了我一些注意事项以及钥匙放置的地点。放完东西整理好部分必需品后我已十分疲惫，床单都没有铺连被子也没有，用衣服枕着头一会就睡着了。第二天大概十一点左右房东来到公寓和我签好协议后我便算是正式开始了独居的生活。一个月750的基本房租不包水电，对于没收入的我算是一笔不算小的支出，所以我尽量从简，这一年来连房间的灯打开的次数都屈指可数，照明全靠从淘宝购买的酷毙灯，也不会放弃在学校机房充满手机电池和充电宝的机会。平均下来月租在850左右吧。有时候天实在太热会开一开空调，但房租也没有超过1000过。

那一年我有更多的时间待在外面，我去了更多金华的地方，也试着做过倒卖二手iPhone（结果并没有盈利反而最后还亏了几百块）。可能是因为短时间内经历了太多，我现在回看照片后竟记不起发生过的一些事，但他们确确实实发生过。

说长不长说短不短，这一年很快就过去了，但这一年发生的事情足够在我心里烙下够深的印记。

关于生活

我不是一个完美的人，我有很多古怪的毛病，古怪的情绪。我也爱幻想，爱做梦。我也开始慢慢接触这个社会，接受现实的生活。暑假离家之前一个星期我就开始有了恋家的情绪。而我到现在依旧没有找到合适的解决办法，只是等待着它的到来然后老老实实接受。

Github pages+Hexo

Posted on August 13, 2017


以下为自己搭建此博客时遇到的一些问题

“如果有什么问题不明白为什么不先去问问神奇海螺呢?”——海绵宝宝

Hexo 配置方面

  • 具体可参照官方文档

  • 安装淘宝源镜像的npm来替代官方源npm

    $ npm install -g cnpm --registry=https://registry.npm.taobao.org

    回车

    安装hexo软件包

    $ cnpm install hexo

    接着本地安装Hexo

    输入

    $ npm install -g hexo

_config.yml下部署deploy

deploy:

- type: git

- repo: https://github.com/你的id/你的id.github.io.git

- branch: master

补充

执行$ hexo d出现Error deployer not found:git的处理方法

$ npm install hexo-deployer-git --save
